# E2.4 — Golden-Path App Onboarding Template

- **Phase:** 2 — Provisioning engine
- **Status:** TODO
- **Depends on:** E1.2, E1.3, E1.5, E1.7, E1.8 (the steps reference these)
- **Pairs with:** E3.1 (established during the Nextcloud epic)
- **Blocks:** E3.3+ (each later app is an instance of this)

## Goal
Make App 2…N cheap by templating the steps.

## Scope
A repeatable checklist — deploy via Helm → wire into SSO → attach DB/storage → configure backups
→ add to monitoring → document. Established *during* the Nextcloud epic (E3.1), then reused.

## Deliverable
The golden-path template itself — a concrete, ordered checklist (with the decisions/inputs each
step needs) that turns "add a new app" into a fill-in-the-blanks exercise.

## Definition of Done
- [ ] Ordered onboarding checklist written.
- [ ] Each step lists required inputs/decisions.
- [ ] Validated by applying it to a second app (not just Nextcloud).
